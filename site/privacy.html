Privacy policy

Last updated: May 11, 2026. This page describes the InstaCleanser Chrome extension (the “extension”). InstaCleanser is an independent open-source project and is not affiliated with Meta, Instagram, or Google.

Summary

  • We do not run a backend service for the extension. There is no InstaCleanser account or login on our servers.
  • Analysis and actions happen in your browser, using the Instagram website session in your active tab.
  • The extension may save recent results and UI state locally on your device using Chrome’s storage API.
  • We do not sell your personal information.

What the extension does

InstaCleanser helps you compare followers and following lists and, when you use it on your own profile, may send unfollow requests the same way the Instagram site does. It operates only while you use it; it does not replace Instagram’s own terms or policies, which you remain subject to when using instagram.com.

Data we do not collect

The extension does not send your Instagram credentials, follower lists, or analysis results to servers operated by the InstaCleanser developers. There is no separate InstaCleanser cloud database tied to this extension.

Data stored on your device

The extension may store a cache of your last successful analysis (for example counts, usernames returned by Instagram’s endpoints, and related UI state) using chrome.storage.local. That data stays on your computer unless you remove it (for example by clearing extension data in Chrome, uninstalling the extension, or using any clear-data action the extension provides in the future).

Instagram and network requests

When you run analysis or use unfollow features, the extension interacts with instagram.com in the browser context you already have open. Those requests are subject to Instagram’s infrastructure and policies. We do not control Instagram’s logging, rate limits, or changes to their site.

Permissions (why they exist)

  • Host access to instagram.com — so the extension can run requests in the page context aligned with your logged-in session.
  • activeTab and scripting — so analysis and actions can run on the Instagram tab you are using.
  • storage — to cache your last run locally for convenience.
  • sidePanel — optional Chrome side panel integration.

Children

The extension is not directed at children under 13. If you are a parent or guardian and believe your child has used the extension in a way that concerns you, you can uninstall it from Chrome and clear extension data.

Changes

If the extension’s behavior or data practices change in a material way, this page will be updated and the “Last updated” date at the top will change. Continued use after an update means you accept the revised policy.
